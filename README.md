Pieces
======

Description
-----------
Othello programmed in Java as project for school.

Features
--------
* Two players (humans, AI)
* 3 difficulties : easy, medium, hard
* Multiplayer (same PC / online)
* Visual themes
* Animations
* and more...(?)

Tools used
----------
* Eclipse : code
* Blender : animation sequence for the pieces
* Gimp : artwork
